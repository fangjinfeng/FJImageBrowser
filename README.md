# FJImageBrowser

具体详见:[简书链接](http://www.jianshu.com/p/57c94ab121c3)

图片浏览器 :上下拖动、横竖屏旋转、支持加载过程先居中，加载完成后放大和直接放大两种效果、支持本地图片和网络图片自动判断。
